OpenNIC JSON API

This is an unofficial API which provides details on OpenNIC DNS servers and TLDs. The information is sourced from the OpenNIC root zone by querying around every 5 minutes. Uptime of servers is calculated every 5 minutes from all past and present checks. You can sort Tier 2 servers by distance from your IP address. The location details of the servers come from their LOC records while your location data comes from GeoIP. Please explore the options above for more details.

Available at:

  • opennicjson.bit
  • opennicjson.mcalinden.me.uk
